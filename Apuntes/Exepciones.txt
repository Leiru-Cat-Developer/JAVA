		Â¿QUE ES UNA EXEPCION?

CUANDO UN PROGRAMA JAVA VIOLA LAS RESTRICCIONES SEMANTICAS DEL LENGUAJE
(SE PRODUCE UN ERROR), LA MAQUINA VIRTUAL JAVA COMUNICA ESTE HECHO AL 
PROGRAMA MEDIANTE UNA EXEPCION.

MUCHAS CLASES DE ERRORES PUEDEN PROVOCAR UNA EXEPCION, DESDE UN 
DESBORDAMIENTO DE MEMORIA O UN DISCO DURO ESTROPEADO, UN INTENTO DE
DIVIDIR POR CERO O INTENTAR ACCEDER A UN VECTOR FUERA DE SUS LIMITES.
CUANDO ESTO OCURRE, LA MAQUINA VIRTUAL JAVA CREA UN OBJETO DE LA CLASE
EXEPTION.
