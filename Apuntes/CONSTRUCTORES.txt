Â¿QUE SON LOS CONSTRUCTURES?

Un constructor es un metodo especial de una clase que se invoca
siempre que se crea un objeto de esa clase.

CUANDO SE CREA UN OBJETO OCURREN 3 COSAS:

-> SE ASIGNA MEMORIA PARA EL OBJETO
-> SE INCIALIZAN LOS ATRIBUTOS DE ESE OBJETO
-> SE INVOCA EL CONSTRUCTOR DE LA CLASE QUE PUEDE SER UNO ENTRE VARIOS

POR EJEMPLO:

PERSONA P1 = NEW PERSONA();

P1 = SE ASIGNA MEMORIA
PERSONA = SE INICALIZAN ATRIBUTOS
NEW PERSONA() = METODO CONSTRUCTOR

CARACTERISTICAS DE LOS CONSTRUCTORES

-> TIENEN EL MISMO NOMBRE DE LA CLASE
-> NO DEVUELVE NINGUN VALOR
-> DEBE DECLARARSE COMO PUBLICO
